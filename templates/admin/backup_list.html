Docker Restore Instructions

To restore a backup in your Docker environment:

Command Line Restore:
# Copy backup file to Docker container
docker cp backup_file.sql container_name:/tmp/

# Restore the backup
docker-compose exec db psql -U dynamic_user -d dynamic_servers -f /tmp/backup_file.sql

# Or restore directly from host
cat backup_file.sql | docker-compose exec -T db psql -U dynamic_user -d dynamic_servers
Fresh Database Setup:
# Stop services
docker-compose down

# Remove old database volume
docker volume rm dynamic-servers_postgres_data

# Start services
docker-compose up -d

# Wait for database to be ready
sleep 10

# Restore backup
cat backup_file.sql | docker-compose exec -T db psql -U dynamic_user -d dynamic_servers
Important: Make sure to stop the web application before restoring to avoid database conflicts.
