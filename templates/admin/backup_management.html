Backup Information
System Database Backups:
  • Contains all user accounts, server configurations, and system data
  • Created on-demand via "System Backup" button
  • Stored in static/backups/system/ directory
  • Use for disaster recovery and system migration
Client Server Backups:
  • Contains client database backups from managed servers
  • Created via "Backup" button on server operations page
  • Stored in static/backups/servers/ directory
  • Downloaded from client servers and stored centrally
Docker Restore (System):
cat system_backup.sql | docker-compose exec -T db psql -U dynamic_user -d dynamic_servers
Client Server Restore:
scp backup.sql user@server:/tmp/
ssh user@server "psql -d database -f /tmp/backup.sql"
